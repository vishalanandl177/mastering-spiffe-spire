# Lab 2 — Debugging SPIRE Issues

This lab walks through common production issues:

1. **Workload gets no SVID** — wrong selector in registration entry
2. **Agent cannot attest** — expired join token or missing service account
3. **Certificate verification fails** — clock skew between nodes
4. **High attestation latency** — datastore performance issue

See `RUNBOOK_SVID_ROTATION_FAILURE.md` and `RUNBOOK_NODE_ATTESTATION.md` for step-by-step debugging.
